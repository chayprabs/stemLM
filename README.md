# stemLM

This project now runs as a Next.js + TypeScript web app.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates the production build.
- `npm run start` runs the production server.

## What changed

- The original static landing page was rebuilt with the Next.js App Router.
- The waitlist CTA now submits through a real API route.
- A functional `/workspace` demo route lets users generate guided STEM walkthroughs, save sessions locally, and export their outline.
